# Fusion - Backend Django

Backend do projeto **Fusion**, desenvolvido como desafio avançado do curso de Django na Udemy.

Responsabilidades:

- Configuração e conexão com **PostgreSQL (Neon)**
- Modelos, migrations e lógica do **Django admin**
- Autenticação de superuser
- Deploy funcional no **Render**
- API para formulários de contato

## Tecnologias

- Python 3.11
- Django 4.x
- PostgreSQL
- Render (deploy)
